Ayuda para instalacion, descargas, cuentas y suscripciones.

Instalacion

La extension funciona en Google Chrome con Telegram Web A y Telegram Web K. Recarga Telegram despues de instalar o actualizar la extension.

Contenido compatible

La extension trabaja con contenido que ya esta visible y accesible para la cuenta del usuario en Telegram Web.

Descargas y progreso

Los videos pueden requerir unos segundos de preparacion. Mantenga Telegram abierto hasta que el progreso finalice y Chrome confirme la descarga.

Cuenta y Premium

Use en la extension el mismo correo utilizado durante la compra. Si el plan no se actualiza, cierre y vuelva a abrir el popup.

Solicitar ayuda

Escriba a encinadownloader@gmail.com e incluya:

  • Version de Chrome y de la extension.
  • Telegram Web A o Telegram Web K.
  • Descripcion breve del problema.
  • Captura del error sin informacion privada.

Privacidad

No envie contrasenas, codigos de autenticacion, datos bancarios ni contenido privado por correo.
